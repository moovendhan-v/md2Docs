# MD to Docs for VS Code

Convert Markdown files into professional, beautifully styled Word (.docx) documents and PDFs directly from VS Code. 

This extension shares the rendering styles, markdown parser, and template engine with the **MD → Docs** web app, ensuring exact consistency between the web editor and your local workspace.

## Features

- **Live HTML Preview**: View your rendered document side-by-side as you type.
- **Save as Word (.docx)**: Generate clean, fully native Word documents styled with one of the curated design templates.
- **Save as PDF**: Open print preview layout and save to PDF instantly.
- **Quick Access Editor Icons**: One-click actions in the top-right corner of any Markdown file.

## Curated Design Templates

Supports all the default styles:
- **Boardroom**: Formal sans-serif with navy accents, ideal for business reports.
- **Academic**: Elegant serif with a centered title and page numbering, perfect for essays.
- **Technical**: Monospace heading style with a clean border.
- **Slate Graphite**: Deep gray accents for structured documentation.
- **Warm Sunset**: Friendly, warm hues.

## How to Use

1. Open any `.md` file in VS Code.
2. Click one of the new icons in the editor toolbar (top-right corner):
   - 👁 **Open Preview**
   - 📄 **Save as Word (.docx)**
   - 📋 **Save as PDF**
3. Alternatively, right-click any Markdown file in the explorer side-bar to choose an action.
